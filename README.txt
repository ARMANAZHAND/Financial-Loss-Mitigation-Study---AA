# Financial Loss Mitigation Study

## Objective

To build a model that predicts for a binary classification task while also minimizing financial losses. Financial losses are incurred via misclassifications – with different losses associated with a false positive than a false negative.

## Methodology

### Data Preparation

- Imputation
- Normalization
- One-hot encoding

### Model Building

- Cross validation
- XGBoost
- Random Forest
- Logistic Regression
- GridSearch parameter tuning

### Performance

- ROC Curves
- Confusion matrices
- Classification reports
- Optimizing for loss
